This directory holds the Generic Security Service Library (GSSLib). See INSTALL for installation instructions, and doc/gss.{info,ps,pdf} for the manual.

Unless you want to use this library to implement a new GSS mechanism, you will need to first install Shishi, as it is the only supported mechanism right now. See http://www.gnu.org/software/shishi/
